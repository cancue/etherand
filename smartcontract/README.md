# etherand

### Prerequisites
- [truffle](https://github.com/trufflesuite/truffle)
- [ganache](https://github.com/trufflesuite/ganache-cli)
- [solhint](https://github.com/protofire/solhint)

```
ganache-cli
truffle migrate
```

### Unit Test
```
truffle test
```

### Lints

solhint contracts/*.sol
